About ShopKart

ShopKart is the Capstone Project for the Thiranex Web Development Internship, built by Mokkapati Kavyasri.

This project demonstrates professional-grade frontend architecture including client-side routing, modular JavaScript, async API calls, state management, and responsive design.

Data is fetched live from the FakeStore API — a free public REST API for e-commerce.
